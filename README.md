# Hi there, I'm Raul Baez Camarillo 👋 

### 🏗️ Architecting Scalable & Clean Ecosystems

I am a Senior Software Architect focused on solving the complexity of large-scale applications through standardized architectural patterns. Currently, I'm building **HexaGen**, a cross-language ecosystem of scaffolding tools designed to enforce **Hexagonal Architecture** and **SOLID** principles.

---

## 🛠️ The HexaGen Ecosystem

My mission is to eliminate "spaghetti code" by providing professional-grade generators for the most popular frameworks.

If you believe in Clean Architecture, please consider starring the HexaGen repos below!
